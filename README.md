# Gemini Chat Navigator 🚀

一个专为 Gemini (gemini.google.com) 网页端打造的聊天历史记录快速定位插件。

## ✨ 功能特点
- **微型刻度条：** 默认收纳在网页右侧边缘，不占用屏幕空间（图1样式）。
- **悬停展开：** 鼠标移入时自动展开为带有文本省略号的话题列表（图2样式）。
- **平滑跳转：** 点击任意话题，页面会自动平滑滚动到对应的聊天位置。
- **自动去重：** 完美解决由于网页底层嵌套导致的标题重复显示 Bug。

## 🛠️ 安装方法（无须上架商店）

由于本插件为开源开发者版本，请按照以下步骤手动安装：

1. 点击本页面右上角的绿色 **`Code`** 按钮，选择 **`Download ZIP`**。
2. 下载完成后，将 ZIP 压缩包解压到你电脑的一个固定位置。
3. 打开你的浏览器（Chrome / Edge 等 Chromium 内核浏览器）。
4. 在地址栏输入并前往：
   - **Chrome:** `chrome://extensions/`
   - **Edge:** `edge://extensions/`
5. 在扩展程序管理页面的右上角，打开 **“开发者模式” (Developer mode)** 开关。
6. 点击左上角的 **“加载已解压的扩展程序” (Load unpacked)** 按钮。
7. 选择你刚刚解压出来的文件夹，即可完成安装！
